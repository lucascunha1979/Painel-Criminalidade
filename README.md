# Painel de Criminalidade — Rio Grande do Sul

Painel interativo para consulta de indicadores criminais do RS por Estado, COREDE e município.

As taxas são calculadas dinamicamente:
- indicadores gerais: casos / população * 100.000;
- Furto de Veículo e Roubo de Veículo: casos / frota * 100.000;
- Abigeato: apenas casos absolutos.

População e frota disponíveis até 2025; para 2026 o painel apresenta casos, mas não taxas.
